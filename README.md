⏱️ Architect Time Tracker

A simple HTML-based tool designed for independent architects to track the time dedicated to each project.

From planning and client calls to research and project development, this application helps organize work sessions and calculate the total time invested in every task.

✨ Overview

Independent architects often work on multiple projects at the same time, dividing their day into small sessions of work:

📐 Planning
📞 Client calls
🔍 Research
🖥️ Design development
📄 Documentation

Keeping track of those hours manually can quickly become difficult.

This project was created as a lightweight and practical solution:
a simple timer that records work sessions and automatically calculates the total hours spent per project.

🚀 Features
⏳ Start and stop timer sessions
📁 Organize hours by project
📊 Automatic total time calculation
📤 Export data to Excel
🖥️ Clean and minimal interface
⚡ Built entirely with HTML, CSS, and JavaScript
🛠️ Built With
HTML5
CSS3
JavaScript
🎯 Goal

The purpose of this tool is to provide architects with an easy way to answer an important question:

"How much time did I spend on this project?"

Without unnecessary complexity or heavy project-management software.
